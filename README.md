# MEAN Ecommerce

## Technologies and Tools

- TextEditor - Visual Studio Code
- VSCode Extensions
  - Prettier - for code formatting
- Development OS - Window 10
- Frontend - Angular
- Backend - NodeJS(ExpressJS)

## Usage

```cmd
git clone https://github.com/skr571999/MEANEcommerce.git

cd client
npm install
npm start

cd server
npm install
npm start
```
